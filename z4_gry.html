1)Kiedy powstał JavaScript (podaj datę).
JavaScript jest obiektowym skryptowym językiem programowania. Stworzony został przez firmę Netscape w 1995 roku.

2)Dlaczego jest to język skryptowy
Zamiast tego jest to język skryptowy, ponieważ wykorzystuje przeglądarkę do wykonywania brudnej roboty .
Jeśli wydasz polecenie zastąpienia obrazu innym, JavaScript poinformuje przeglądarkę, aby to zrobiła.

3)Gdzie jest wykonywany JS (klient lub serwer)
JavaScript jest skryptem po stronie klienta.

4)Dlaczego jest JS to bezpieczne narzędziem?
js, podobnie jak inne języki i środowiska, jest bezpieczny, gdy projekty w nim budowane są traktowane z odpowiednią dbałością i zastosowaniem najlepszych praktyk.

5)Czy jest to język obiektowy , jeśli tak to dlaczego.
JavaScript jest obiektowym, ponieważ jest skryptowym językiem programowania.

6)Wymień zalety JS:
-szybkość
-prostota
-tworzyć nowoczesne aplikacje internetowe do bezpośredniej interakcji

7)Framework to:
Framework (platforma programistyczna) – jest to szkielet do budowy aplikacji, który dostarcza niezbędne biblioteki i komponenty oraz definiuje strukturę i działanie danej aplikacji.

8)Czy do aplikacji mobilnych można użyć Framework
Istnieje wiele frameworków, które można wykorzystać do tworzenia aplikacji webowych (stron internetowych),
aplikacji mobilnych (aplikacji na telefon) lub innego oprogramowania.

Strona responsywna dopasowuje się do rozdzielczości wyświetlaczy urządzeń przenośnych,
